## 🚀 Sobre o projeto

Este repositório documenta minha jornada de aprendizado em **T-SQL com SQL Server**, com foco em **análise de dados, manipulação de informações e soluções aplicadas ao mundo real**.

O objetivo é evoluir do básico ao avançado, construindo consultas eficientes e projetos práticos voltados ao dia a dia de um **Analista de Dados**.

---

## 🎯 Objetivos

* Aprender T-SQL do básico ao avançado
* Desenvolver queries eficientes
* Trabalhar com grandes volumes de dados
* Criar **Stored Procedures**, **Views** e processos de **ETL**
* Resolver problemas reais com dados
